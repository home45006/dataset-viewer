## 🔧 完整启动步骤

### 启动前端 (已完成 ✅)
```bash
cd dataset-viewer-frontend
npm install    # 已完成
npm run dev    # ✅ 正在运行
```

### 启动后端 (需要 Rust)
```bash
# 1. 安装 Rust (如果未安装)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source ~/.cargo/env

# 2. 启动后端服务
cd dataset-viewer-backend
cargo run --bin server
```
